# YourDebtRelief Calculator

3 calculators showing savings through YourDebtRelief vs other options

## Run Locally

Clone the project

```bash
  git clone https://github.com/jthurber87/YourDebtRelief
```

Go to the project directory

```bash
  cd debt-calculator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
